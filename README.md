JEE Advanced Question Extraction

This project extracts questions from JEE Advanced examination papers and converts them into a structured format. JEE Advanced papers are usually available as PDF documents, which makes it difficult to automatically analyze or organize the questions.

The goal of this project is to process those PDFs and extract useful information such as question numbers, question statements, and multiple-choice options so they can be stored as structured data.


Extracts text from JEE Advanced PDF papers
Identifies question numbers and question statements
Parses multiple-choice options
Stores extracted data in structured format such as JSON or CSV

Technologies Used
Python
PDF parsing libraries (such as PyPDF2 or pdfplumber)
Regular Expressions for detecting question patterns
JSON / CSV for structured data storage
